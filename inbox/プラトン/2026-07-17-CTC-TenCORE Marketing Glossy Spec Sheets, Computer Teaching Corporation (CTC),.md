TenCORE* Language Authoring System

The right tool for the job ...

The TenCORE Language Authoring System offers a proven production environment for the professional courseware developer.
It’s a PC-based system that allows trainers and educators to efficiently translate ideas into high quality, individualized instruction.

You’ll appreciate the menu-driven organization of TenCORE’s authoring utilities. Optimized for creating computer-based
courseware, our integrated editing tools help you quickly combine text and graphics, interactive video, response based
branching, and comprehensive student management. Detailed system documentation assists both first-time authors and
experienced developers.

TenCORE speaks your language

Designed to meet the needs of the most demanding courseware developer, TenCORE’s powerful commands are based on
common instructional terminology, which means that you and TenCORE already speak the same language. This provides you
the freedom to build courseware exactly the way you imagined it, without compromising your original instructional design.

TenCORE lessons are organized into named units containing text and graphics,
answer judging, student feedback, and branching instructions. This integration
allows you to immediately try out ideas, speeding up development and revision.

Prompted editors

The full-screen command editor combines pull-down menus and a complete range
of text editing features to speed up the creation and editing of TenCORE lessons.

Global search/replace, cut, paste, copy, import/export, and more are provided. And,
whenever you need it, complete on-line documentation for the entire TenCORE
command language is just a keypress away.

The object oriented graphics editor lets you lay out text, drawings, images, and
video exactly as the student will see them. The editor automatically writes the
program statements needed to recreate your screen design, giving you access to
the full breadth and power of the TenCORE language.

An image editor is provided which lets you “paint” pictures directly within TenCORE
and modify images imported from PC Paintbrush® and Dr. Halo® picture files.

Other editing tools help you create custom character fonts, roster students and
monitor their progress, and manage TenCORE lesson and data files.

Power and Flexibility

TenCORE makes it easy for you to create high quality courseware with graphic
features such as windowing, object oriented or bit-mapped graphics, sizing and
rotation of text or graphics, and interactive video with graphic overlay. It’s perfect
for creating sophisticated simulations which include animated graphics, complex
calculations, real-time response analysis, and flexible lesson branching.

Open-ended

The TenCORE Language Authoring System has been designed to adapt to your
growing needs. The system allows you to easily run external applications by
providing direct access to DOS files and memory. It also offers user control over
a variety of peripherals including digital audio boards, CD-ROM, and simulation
devices. TenCORE comes on both 5 l M n and 3 x k” diskettes.
