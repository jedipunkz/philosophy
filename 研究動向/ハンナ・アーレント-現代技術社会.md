# ハンナ・アーレント-現代技術社会

## 動向

アーレントの『人間の条件』は、古代政治への郷愁としてではなく、自動化、ベーシックインカム、SNS、気候危機を考える理論として再利用されている。中心にあるのは、労働からの解放がただちに自由を意味しないという問題である。

Riley Hannah Muriel Lewicki の “The Problem of Idleness” は、自動化と UBI をアーレントの活動類型から検討する。労働から解放された時間が、活動・制作・思考・公共的参加へ向かわず、消費の循環へ吸収されるなら、意味の危機が生じる。したがって UBI は所得保障だけでなく、公共世界への参加条件と合わせて考える必要がある。

SNS 研究では、デジタル空間が公共圏に見えながら、実際には私的なものと公共的なものを混合する「社会的なもの」として機能する点が論じられる。気候危機研究では、現実否認とイデオロギー的閉鎖性が、全体主義論の現代的適用先として扱われる。

## 論点

- 自動化による労働からの解放は、公共的自由の条件にはなりうるが、それ自体では自由ではない。
- UBI 論は、所得保障だけでなく、余暇が消費へ回収される危険を考慮する必要がある。
- SNS は公共圏の代替ではなく、公私境界を曖昧にする社会的領域として分析される。
- 気候危機への応答不能は、現実否認とイデオロギーの問題としてアーレント的に読まれる。
