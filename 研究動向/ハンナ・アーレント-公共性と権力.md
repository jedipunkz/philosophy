# ハンナ・アーレント-公共性と権力

## 動向

現代研究では、アーレントの公共性論と権力論が、暴力・同意・非暴力・デジタル公共圏の問題へ接続されている。焦点は、公共圏を単なる「意見交換の場」としてではなく、自由で平等な人々が共通世界のもとで現れる条件として捉える点にある。

Gail M. Presbey の “Hannah Arendt on Power, Consent, and Coercion” は、アーレントの権力概念をガンディーの非暴力論と比較する。アーレントにおいて権力は、強制や暴力ではなく、人々が自由に同意し共同で行為する能力である。暴力は一時的服従を生むが、自由な協力を作れないため、安定した権力の代替にはならない。

Metehan Karakurt と Aykut Aykutalp の SNS 論は、ソーシャルメディアをアーレント的公共圏と見なせるかを問う。SNS は政治的発言や抗議運動の場になる一方、私的・親密な出来事を可視化し、公私の区別を曖昧にする「社会的なもの」の場でもある。そのため、単に発言が公開されることと、共通世界をめぐる公共的行為とは区別される。

## 論点

- 権力は暴力や強制ではなく、共同的行為と自由な同意に依存する。
- プロパガンダや操作は、支配者が真の同意を得られていないことの徴候として読まれる。
- 公共圏は「見られる場所」ではなく、共通世界を共有し、差異ある人々が行為と言葉で現れる場である。
- SNS は公共的議論を可能にするが、同時に私的なものを公開化し、公共性を「社会的なもの」へ変質させる。
