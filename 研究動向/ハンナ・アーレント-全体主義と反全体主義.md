# ハンナ・アーレント-全体主義と反全体主義

## 動向

`inbox/` のアーレント関連資料では、全体主義論は歴史的ナチズム・スターリニズムの分析に限定されず、現代政治の診断概念として再利用されている。ただし、その再利用は単純な「現代は全体主義化している」という主張ではない。むしろ、アーレントの全体主義概念をどこまで拡張できるか、また拡張するときに何を見失うかが問われている。

Kei Hiruta の “A democratic consensus?” は、反全体主義思想を一枚岩の戦後民主主義コンセンサスとして読むことに警戒する。アイザイア・バーリンとアーレントはともにナチズムとスターリニズムに反対したが、バーリンの自由主義的多元論と、アーレントの公共的行為・複数性の政治哲学は一致しない。したがって、アーレントの全体主義批判は自由主義一般へ回収できない。

Larry Alan Busk の “What Is ‘Totalitarian’ Today?” は、気候危機を背景に、全体主義を「現実否認」「イデオロギー体系の経験的証拠への優越」「無力感と全能感の併存」として読み直す。ここでは全体主義ラベルは外部の敵だけでなく、気候破局へ対応できない政治経済システムそのものへ向け返される。

## 論点

- 反全体主義は共通の政治哲学ではなく、内部に複数の相容れない立場を含む。
- アーレントの全体主義論は、孤立した大衆、イデオロギー、現実否認、プロパガンダを分析する道具として現代化されている。
- 気候危機への応用では、全体主義を例外的な体制ではなく、支配的制度が現実を否認する構造として読む方向がある。
- ただし、この拡張は概念の強度を弱める危険もあるため、歴史的全体主義との連続性と断絶を区別する必要がある。
