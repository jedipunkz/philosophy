# 親鸞『顕浄土真実教行証文類』（教行信証）

| 項目 | 内容 |

| 著者 | 親鸞（1173–1263） |

| 地域 | 日本、東洋哲学 |
| 分野 | 浄土仏教、日本仏教、宗教哲学 |
| 主題 | 阿弥陀如来の本願力（他力）による凡夫の救済を、経・論・釈の引用を積み上げて論証し、浄土真宗を「真実の教え」として顕らかにする。 |
| キーワード | 教行信証、他力本願、絶対他力、悪人正機、信心正因称名報恩、六巻構成、正信念仏偈 |

『顕浄土真実教行証文類』（けんじょうどしんじつきょうぎょうしょうもんるい）は、鎌倉時代前半の僧・親鸞の主著であり、浄土真宗の**根本聖典**である。全6巻からなり、崇敬の念をこめて本願寺派では「御本典」、大谷派では「御本書」と呼ばれる。真宗十派（真宗教団連合）は、この書の草稿本が完成した元仁元年（1224年）4月15日をもって「立教開宗」とし、同日を「真宗立教開宗記念日」と定めている。表題を素直に読めば「浄土（＝阿弥陀の救い）の真実を、教・行・証（さらに信）という文の類によって顕らかにする書」であり、親鸞が生涯をかけて磨き上げた思想の体系書といえる。

## 師・法然の教えを「論証」する書

親鸞自身に新しい宗派を開く意志はなかった。彼が生涯の喜びとしたのは、専修念仏を説く法然に師事できたことであり、著書『高僧和讃』でも「浄土真宗ひらきつゝ」と讃えられるべきは自分ではなく師・法然だと明言している。したがって『教行信証』は、法然の主著『選択本願念仏集』（『選択集』）の**解説書であると同時に、その正当性を経典・論釈によって論証した文書**という性格をもつ。

その論証のスタイルは独特で、親鸞自身の言葉よりも、拠り所となる経典（浄土三部経）と、インド・中国・日本の**七高僧**（龍樹・天親・曇鸞・道綽・善導・源信・源空〈法然〉）の論釈からの引用を膨大に積み上げ、その配列と要所の按文によって思想を立ち上げていく。冒頭で釈尊の出世本懐の経とされる『大無量寿経』（『大経』）を「真実の教」と位置づけ、阿弥陀如来の本願（四十八願）と、本願から与えられる名号「南無阿弥陀仏」こそが浄土門の真実の教え＝「浄土真宗」であると示すのが、全巻を貫く骨格である。

## 六巻の構成

全6巻は「教・行・信・証・真仏土・化身土」の6つの文類からなり、それぞれ冒頭・巻末に序が置かれる。

- **総序** — 全体の序文。
- **教巻**（顕浄土真実教文類一） — 『大無量寿経』を真実の教とする根拠を示す。
- **行巻**（顕浄土真実行文類二） — 名号（南無阿弥陀仏）を称える「行」が阿弥陀の回向したものであることを説く。巻末に**「正信念仏偈」（正信偈）**が置かれる。
- **信巻**（顕浄土真実信文類三） — 別序を含む。往生の正因である「信心」を論じる、思想的な中核。
- **証巻**（顕浄土真実証文類四） — 信によって得られる証（さとり・涅槃）を説く。
- **真仏土巻**（顕浄土真仏土文類五） — 阿弥陀仏とその浄土の真実性を論じる。
- **化身土巻**（顕浄土方便化身土文類六、本・末） — 方便としての仏土や、聖道門・自力諸行との対比を扱う。巻末の「竊かに以みれば聖道の諸教は行証久しく廃れ…」以降は「後序」と呼ばれ、親鸞自身の来歴（承元の法難、法然からの『選択集』書写許可など）が記される貴重な部分である。

とりわけ「行巻」末の**正信念仏偈**は、七言120句からなる偈頌で、阿弥陀の本願と七高僧の教えを凝縮して讃える。真宗門徒の日常勤行に用いられ、この主著のなかで最も広く親しまれた部分である。

## 中心思想 ― 絶対他力と信心正因

『教行信証』が説く救済観は、他の仏教宗派と大きく異なる。

- **他力本願・絶対他力** — 往生は阿弥陀如来の「本願力」（他力）のはたらきによって成就するのであり、凡夫のはからい（自力）によるのではない。自力で救われようとする心が完全に消えた「信心決定」のその瞬間に、臨終を待たず往生が定まるとする。なお、後に真宗を象徴する語となる「絶対他力」という用語自体は、親鸞の著作には一度も現れない。
- **信心正因・称名報恩** — 往生の正しい因（正因）は「信心」であって、口に称える念仏（称名）ではない。念仏は、往生が定まったのちに「我が名を称えよ」という阿弥陀の願い（第十八願）に応える**報恩感謝の行**として位置づけられる。念仏を往生のための修行・善行とは捉えない点に、法然からさらに一歩を進めた徹底がある。
- **本願招喚の勅命** — 親鸞は名号「南無阿弥陀仏」を、「疑いなく我をたのみ、我が国に生まれんと思え」という阿弥陀からの**呼びかけ**と理解した。信も行もすべて阿弥陀の側が用意し、そのまま与えるという構図である。
- **往還二回向** — 阿弥陀のはたらきには、衆生を浄土へ往生させる「往相回向」と、浄土から再びこの世に還って人々を救う「還相回向」の二側面があるとする。

これらの延長線上に、真宗を最も象徴する人間観である**悪人正機**（「善人なおもて往生を遂ぐ、況んや悪人をや」）が立つ。自力で善をなせると思う「善人」よりも、自らの弱さと罪深さに向き合う「悪人＝凡夫」こそが阿弥陀の本願の救いの本来の対象だという逆説である（この主題を最も鮮烈に伝えるのは、親鸞自身の著ではなく弟子・唯円の編とされる『歎異抄』である）。

## 「非仏説」論争を呼ぶ独自性

信心一つで、念仏さえ往生の因とせず、戒律も自力の修行も要しないという親鸞の教義は、初期仏教はもちろん大乗仏教のなかでも際立って特異である。仏教学者ポール・ウィリアムズが「親鸞の教義体系はとても良い。しかし、それは仏教だろうか？」と評したように、その解釈を「非仏説」ではないかと問う指摘は古くからある。人間の罪深さと絶対者による救済という構図から、キリスト教の信仰義認に近いとする見方もあり、実際に西田幾多郎・吉本隆明・滝沢克己・カール・バルトらの思想圏で親鸞は繰り返し参照されてきた。一方で、これを仏教の枠内で理解しようとすることを戒め、仏教伝来以前から現代まで通底する日本の精神的土壌が仏教を通して顕現したものとして積極的に評価する立場もある。

## 唯一の真跡本「坂東本」

『教行信証』は、親鸞真跡（自筆）本が現存する点でも貴重である。真宗大谷派が所蔵する一本がそれで、門弟・性信を開基とする坂東報恩寺に伝わったことから**「坂東本」**（『御草本』とも）と通称される。東国在住時代に草稿が成り、最晩年まで推敲が重ねられたもので、朱筆・墨による加筆が随所に残る。昭和27年（1952年）に国宝指定。関東大震災の際、収蔵先の浅草別院が焼失したにもかかわらず、金庫内の酸素が焼失前に消費されたことで焼亡を免れたという逸話でも知られる。2003–2004年の修復調査では、竹などを尖らせた角筆による約700か所の書き込み（うち約400か所は重要箇所への「合点」）が新たに発見され、親鸞がこの書をいかに繰り返し読み込み推敲したかを物語っている。

## 依拠する経典・論釈

親鸞が拠り所とした根本経典は「浄土三部経」——『仏説無量寿経』『仏説観無量寿経』『仏説阿弥陀経』——であり、なかでも『仏説無量寿経』を『大無量寿経』『大経』と呼んで最重要視した。加えて、他力念仏の系譜を継ぐ七高僧の論釈、すなわち龍樹『十住毘婆沙論』（易行品）、天親『浄土論（往生論）』、曇鸞『浄土論註』、道綽『安楽集』、善導『観経疏』、源信『往生要集』、法然『選択本願念仏集』が、本書のなかで縦横に引用・按配されている。

## 位置づけ

『教行信証』は、親鸞の生涯と思索の到達点であり、その後の浄土真宗という日本最大の仏教教団（約2万2千か寺）の思想的基盤となった。親鸞の他の著作——『浄土文類聚鈔』（本書の要点をまとめた書）、『愚禿鈔』、和讃の「三帖和讃」（『浄土和讃』『高僧和讃』『正像末和讃』、国宝）など——は、いずれもこの主著が示した信心の教えを、別の角度から、あるいは平易な和文で展開したものと読むことができる。親鸞の哲学を一冊で問うなら、まずこの書が代表作である。
